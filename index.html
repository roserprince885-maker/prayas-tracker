How to set up Supabase (free, 5 minutes)

  1. Go to supabase.com → click Start your project → sign up free
  2. Click New project → give it a name like prayas-tracker → set a database password → click Create
  3. Wait ~1 minute for it to set up, then go to Project Settings → API
  4. Copy the Project URL and the anon / public key
  5. Now go to the SQL Editor tab in Supabase and paste this SQL, then click Run:
CREATE TABLE IF NOT EXISTS tracker (
  key TEXT PRIMARY KEY,
  value TEXT,
  updated_at TIMESTAMPTZ DEFAULT now()
);

That's it! Come back here, paste your URL and key, and you're synced.
